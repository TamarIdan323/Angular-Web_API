# מערכת לניהול מכירה סינית

פרויקט זה הוא אתר לניהול מכירות, שפותח ב-Angular וב-C# עם בסיס נתונים SQL.

## תכונות עיקריות

- התחברות משתמשים והרשאות
- סל קניות דינמי
- מערכת תשלום מאובטחת
- ממשק ניהול למתנות והטבות

## טכנולוגיות בשימוש

- Angular (Frontend)
- C# (.NET Core) בצד השרת (Backend)
- SQL Server לבסיס הנתונים
